# Contribution Guidelines

Thank you for your interest in contributing to the Checklist Flexo project.

## How to Contribute:

### Reporting Bugs
- Check if the bug has already been reported in the Issues section. If not, create a new issue with a clear description.

### Suggesting Features
- Open an issue with the label enhancement. Describe the feature clearly.

### Submitting Pull Requests
- Fork the repository, create a new branch, make changes, commit with clear messages, push to your fork, and submit a pull request.

### Code Style
- Keep code clean and readable, add comments where necessary, test changes before submitting.

## Contact
- For questions, open an issue or contact the maintainers.
